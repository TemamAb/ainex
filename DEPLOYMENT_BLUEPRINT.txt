Step 2: Prepare Wallet (2 min)
        → Create Ethereum wallet (MetaMask, Ledger, etc.)
        → Get wallet address (0x...)
        → Fund with some ETH (for gas)

Step 3: Log into Render (1 min)
        → https://render.com/dashboard
        → Sign in with GitHub

Step 4: Deploy Service (2 min)
        → Click "New +" → "Web Service"
        → Connect "TemamAb/myneon"
        → Set environment variables
        → Click "Create Web Service"
        → Wait for "Running" status

Step 5: Verify (1 min)
        → Copy service URL
        → Test: /health endpoint
        → Check logs
