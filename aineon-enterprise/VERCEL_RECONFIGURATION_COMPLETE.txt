================================================================================
                        IMPORTANT DIFFERENCES
================================================================================

POLLING VS WEBSOCKET:
  WebSocket (Render):
    - Real-time updates (<10ms)
    - Persistent connection
    - Uses port 8765
    - Scales with connections

  Polling (Vercel):
    - Updates every 1-5 seconds
    - Stateless HTTP requests
    - Automatic scaling
    - Lower latency variance

  RECOMMENDATION: For dashboard, 3-5 second polling is sufficient
  UPDATE FRONTEND: Change POLL_INTERVAL in dashboard.html to 3000ms

STATEFULNESS:
  Render: Can maintain state between requests
  Vercel: Stateless (every request is independent)

  IMPLICATION: Can't store profit data in memory
  SOLUTION: Use external database (Redis, MongoDB)

FILE SYSTEM:
  Render: /tmp persistent, /home available
  Vercel: /tmp ephemeral (can be deleted between requests)

  IMPLICATION: Can't write files permanently
  SOLUTION: Use external storage or environment variables

ENVIRONMENT:
  Render: Linux container (always running)
  Vercel: Node.js + Python functions (event-driven)

  IMPLICATION: Background processes won't work
  SOLUTION: Use Vercel Cron functions for scheduled tasks

TO DEPLOY TO VERCEL:

Step 1: Create Vercel Account
  1. Go to https://vercel.com
  2. Sign up or log in
  3. Click "New Project"
  4. Select "Import from Git"
  5. Connect GitHub account
  6. Select: github.com/TemamAb/ainex

Step 2: Set Environment Variables
  In Vercel Dashboard ‚Üí Project Settings ‚Üí Environment Variables:
  
  ALCHEMY_API_KEY = <your-key>
  INFURA_API_KEY = <your-key>
  PRIVATE_KEY = <your-key>
  WITHDRAWAL_ADDRESS = <wallet-address>
  AUTO_WITHDRAWAL_ENABLED = false
  MIN_PROFIT_THRESHOLD = 0.5
  ETH_PRICE_USD = 2850.0
  ENVIRONMENT = production
  LOG_LEVEL = INFO

Step 3: Deploy
  Vercel automatically builds when you push to main:
  
  git push origin main
  
  Or manually deploy from Vercel Dashboard

Step 4: Test
  Dashboard: https://ainex.vercel.app/dashboard
  API: https://ainex.vercel.app/api/profit
  Health: https://ainex.vercel.app/health
