TROUBLESHOOTING:

Setup script won't run:
  • Check .env file exists: ls -la .env
  • Verify Python: python3 --version
  • Test RPC manually: curl -X POST <your RPC URL>

Deployment fails:
  • Check Docker: docker --version
  • Check Docker Compose: docker-compose --version
  • View logs: docker logs aineon-engine-prod

Monitor shows no profit:
  • Check if system is online: curl http://localhost:8081/status
  • Check for opportunities: curl http://localhost:8081/opportunities
  • Verify account balance: curl http://localhost:8081/profit

Withdrawal fails:
  • Check API health: curl http://localhost:8081/health
  • Check balance is sufficient: curl http://localhost:8081/profit
  • Try again with: curl -X POST http://localhost:8081/withdraw
